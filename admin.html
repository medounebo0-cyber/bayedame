📖 Comment obtenir le Bot Token et Chat ID ?

1. Bot Token

  • Dans Telegram, cherche @BotFather
  • Tape /newbot
  • Donne un nom (ex: Baye Dame Notifs) et un username (ex: bayedame_notifs_bot)
  • BotFather te donne un token comme 123456:ABC-def...

2. Chat ID

  • Dans Telegram, cherche @userinfobot
  • Clique « Démarrer » → il t'affiche ton Id

3. Activer ton bot

  • Cherche le bot que tu viens de créer (par son username)
  • Clique « Démarrer » pour autoriser les messages
